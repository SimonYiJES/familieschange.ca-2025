INTRODUCTION
------------

The vimeo embed field module will create a simple field widget type that allows
to add a vimeo video url to embed videos from Vimeo and show their thumbnail

REQUIREMENTS
-------------

*None. (Other than a clean Drupal 8 installation)


INSTALLATION
-------------

Install this module as usual. Please see
http://drupal.org/documentation/install/modules-themes/modules-8


CONFIGURATION
-------------

Features
--------

Vimeo embed videos sizes.
------------------------

This module has included a responsive option and custom option
to that expand the player to the width of its container as well as custom size.

Vimeo thumbnails displayed with image styles.
---------------------------------------------

Thumbnails can be linked to the video url with new or same window option.


Embed video player configuration
--------------------------------

Size: Having the option in dropdown to change the embed size.
Width: Custom Width of Vimeo player.
Height: Custom Height of Vimeo Player.
Allowautoplay: Checkbox for allowing play automatically the video on load.
Title: Checkbox for allowing display the name of the video.
Loop: Checkbox for allowing play the video repeatedly.
Byline: Checkbox for allowing display who the video is by.
Portrait: Checkbox for allowing display the video submitter's picture or image.
Color: Checkbox for allowing change the color for embed video player icons.

Supported url formats
---------------------

https://vimeo.com/11111111
http://vimeo.com/11111111
https://www.vimeo.com/11111111
http://www.vimeo.com/11111111
https://vimeo.com/channels/11111111
http://vimeo.com/channels/11111111
https://vimeo.com/groups/name/videos/11111111
http://vimeo.com/groups/name/videos/11111111
https://vimeo.com/album/2222222/video/11111111
http://vimeo.com/album/2222222/video/11111111
https://vimeo.com/11111111?param=test
http://vimeo.com/11111111?param=test
